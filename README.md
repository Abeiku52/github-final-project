# GitHub Final Project

## Project Overview
This repository contains the final project for the GitHub UI and Git CLI hands-on labs. The project demonstrates proficiency in using GitHub's web interface and Git command-line tools through a series of practical tasks.

## Project Structure
- **README.md**: Project documentation and overview
- **LICENSE**: Apache 2.0 license file
- **CODE_OF_CONDUCT.md**: Community guidelines and standards
- **CONTRIBUTING.md**: Contribution guidelines for the project
- **simple-interest.sh**: Bash script calculator for simple interest computation

## Tasks Completed
This project fulfills 9 tasks across two main categories:

### Part 1: GitHub UI (12 points)
1. ✅ README.md with project details
2. ✅ Apache 2.0 LICENSE file
3. ✅ CODE_OF_CONDUCT.md file
4. ✅ CONTRIBUTING.md file
5. ✅ simple-interest.sh calculator script

### Part 2: Git CLI (8 points)
6. ✅ Repository forking demonstration
7. ✅ Branch merging operations
8. ✅ Pull request validation
9. ✅ Branch status verification

## Getting Started
To use the simple interest calculator:
```bash
chmod +x simple-interest.sh
./simple-interest.sh
```
